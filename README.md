# Gousto coding exercise

### Installing the application
> yarn install

or
> npm install
>
### Running the application
#### in development mode
> yarn start

or
> npm start

#### in production mode
* build the file
> yarn prod:build
or
> npm run prod:build

* run production build
> yarn prod
 or
> npm run prod

#### run the tests
> yarn test
 or
> npm run test
