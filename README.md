# Gousto coding exercise

### Installing the application

yarn install

or
```javascript
npm install
```
>
### Running the application
#### in development mode
```javascript
yarn start
```
or
```javascript
npm start
```

#### in production mode
* build the file
```javascript
yarn prod:build
```
or
```javascript
npm run prod:build
```

* run production build
```javascript
yarn prod
```
or
```javascript
npm run prod
```

#### run the tests
```javascript
yarn test
```
or
```javascript
npm run test
```
